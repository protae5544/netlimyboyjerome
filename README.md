วัตถุประสงค์หลัก:
- สแกน QR แล้วต้องเรียก URL ฝั่งเซิร์ฟเวอร์เท่านั้น
- เซิร์ฟเวอร์ต้องสตรีม “ไฟล์ PDF ต้นฉบับ” กลับไปแบบดิบ (ไม่แปลง/ไม่เรนเดอร์ใหม่) เพื่อรักษาหน้าตาเดิม 100%
